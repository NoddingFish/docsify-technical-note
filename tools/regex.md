# 正则表达式

案例：

1. 千分位格式化

   给 `100000000000` 格式化为： `100,000,000,000`

   `Js` 实现

   ```js
   let str = '100000000000';
   
   let res = str.replace(/\B(?=(\d{3})+$)/g, ',');
   
   console.log(res); // 100,000,000,000
   ```

   `PHP` 实现

   ```php
   $str = '100000000000';
   
   $res = preg_replace('/\B(?=(\d{3})+$)/', ',', $str);
   
   echo $res . PHP_EOL; // 100,000,000,000
   ```
